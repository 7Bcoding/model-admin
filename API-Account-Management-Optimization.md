# API账号管理优化说明

## 优化概述

我们对API账号管理的表进行了优化，使供应商信息可以从供应商管理中的name字段选择，提供了更好的管理体验和数据一致性。

## 主要优化内容

### 1. 供应商数据源集成

- **新增供应商管理数据加载**：在API账号管理页面中集成了供应商管理的数据
- **智能数据合并**：将供应商管理中的name字段与现有账户中的vendor字段合并
- **优先级设计**：优先显示供应商管理中的name，确保数据规范性

### 2. 供应商选择器优化

#### 分组显示
- **供应商管理组**：显示来自供应商管理中的name字段，带有⭐标识
- **现有账户组**：显示现有账户中使用的vendor（排除已在供应商管理中的）

#### 用户体验提升
- **可视化标识**：
  - 供应商管理中的name显示蓝色星标图标
  - 现有账户中的vendor显示"已使用"标签
- **帮助提示**：添加了"优先选择供应商管理中的名称，可获得更好的管理体验"的提示
- **保持兼容性**：仍支持手动输入新的供应商名称

### 3. 技术实现

#### 新增API集成
```javascript
import { getVendors } from '@/api/fusion-vendors'
```

#### 数据管理
- 新增 `vendors` 响应式数据存储供应商列表
- 新增 `vendorsLoaded` 标记供应商数据加载状态
- 在 `loadAccounts` 函数中并行加载供应商数据

#### 计算属性优化
- `vendorOptionsFromManagement`：供应商管理中的name选项
- `vendorOptionsFromAccounts`：现有账户中的vendor选项（排除重复）
- `availableVendors`：合并后的所有可用供应商（向后兼容）

### 4. 样式优化

#### 供应商选择器样式
- 分组标题样式优化
- 选项悬停效果
- 帮助提示样式
- 响应式设计支持

#### 视觉层次
- 使用不同颜色区分数据来源
- 图标和标签提供清晰的视觉提示
- 保持与整体设计风格一致

## 使用说明

### 创建新账户时
1. 打开"创建账户"对话框
2. 在"供应商"字段中，会看到分组显示的选项：
   - **供应商管理**：来自供应商管理中的name字段，推荐使用
   - **现有账户**：已在其他账户中使用的vendor
3. 优先选择"供应商管理"组中的选项
4. 也可以手动输入新的供应商名称

### 筛选功能
- 表格头部的供应商筛选器会显示所有可用的供应商
- 包括供应商管理中的name和现有账户中的vendor

## 优势

1. **数据一致性**：通过集成供应商管理，确保供应商信息的统一管理
2. **用户体验**：清晰的分组和标识帮助用户做出更好的选择
3. **向后兼容**：保持对现有数据和工作流程的完全兼容
4. **可扩展性**：为未来的供应商管理功能扩展奠定基础

## 技术细节

### 数据加载流程
1. 页面初始化时并行加载：
   - 账户数据 (`loadAllAccounts`)
   - 模型数据 (`loadAllModels`)
   - 供应商数据 (`loadAllVendors`)

2. 数据处理：
   - 提取供应商管理中的name字段
   - 合并现有账户中的vendor字段
   - 去重并排序

### 性能优化
- 使用计算属性缓存处理结果
- 并行加载数据减少等待时间
- 智能去重避免重复显示
